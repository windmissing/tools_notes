  • 1. 问题现象
  • 2. 问题原因
  • 3. 解决方法

1. 问题现象

2. 问题原因

下载速度太慢，下载时间超时了。
如果没有报这样的错误，只是下载速度太慢，也可以用下面的方法。

3. 解决方法

  1. 打开页面https://pypi.org/

  2. 搜索要下载的库

  1. 选择要下载的库

  1. 选择“Download files”

  1. 根据操作系统和python版本选择合适的库版本下载到本地

  1. pip install 库在本地的地址
